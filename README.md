# Task Tracker Heatmap

A simple interactive heatmap to visualize daily activity or completed tasks throughout the year. Inspired by GitHub contribution graphs.

## Features
- Year-long grid visualization (Monday-start week)
- Color-coded activity levels
- Tooltip on hover showing the count and date
- Easily customizable with your own data source

## Getting Started
1. Clone or download the repository.
2. Open `index.html` in your browser.

## File Overview
- `index.html` — Main web page for the heatmap
- `style.css` — Visual styles for heatmap and page
- `script.js` — Generates the heatmap with mock data for 2025 (replace with your own data!)

## Customization
- Edit `script.js` to load your own task/activity data instead of random mock data.
- Adjust color levels or grid size as needed.
